# 🍽️ RESTAURANT DATA ANALYSIS PROJECT

## 📌 Overview
This project focuses on analyzing restaurant data to answer key business questions and uncover insights that can help improve decision-making.

The analysis is based on two datasets that capture information about the restaurant's menu and customer orders.

---

## 📂 Dataset Description

The project uses two CSV files:

### 1. `menu_items.csv`
This dataset contains information about the items available on the restaurant’s menu, including:
- Item names  
- Categories  
- Prices  
- Other relevant details  

### 2. `order_details.csv`
This dataset contains records of customer orders made in the restaurant, including:
- Order IDs  
- Item IDs  
- Order date and time, etc.

---

## 🎯 Project Objectives

### ✅ Objective 1: Explore the Menu
Analyze the `menu_items` dataset to understand:
- What items are available on the menu  
- Pricing structure  
- Categories of food offered  

---

### ✅ Objective 2: Explore Order Data
Examine the `order_details` dataset to gain insights into:
- Ordering patterns  
- Frequency of orders  
- Data structure and completeness  

---

### ✅ Objective 3: Customer Behavior Analysis
Combine both datasets to understand how customers are reacting to the menu:
- Most and least popular items  
- Revenue-generating items  
- Customer preferences and trends
